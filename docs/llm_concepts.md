# OpenAI SDK

A toolkit that abstracts HTTP requests and responses.

# Client

Connection object used to communicate with LM Studio.

# Temperature

Controls randomness.

0.0 = deterministic

1.5 = creative/random

# Max Tokens

Maximum output length.

# Response Object

Contains:

- model
- usage
- choices
- finish_reason
- content

# LM Studio

Provides OpenAI-compatible local API.
